# Inukta Website

Marketing website for Inukta — One Platform. Endless Possibilities.

Built with React (Vite) and React Router. Pages: Home, Services, Contact,
Why Inukta?, and Book an appointment.

## Run locally

npm install
npm run dev

## Build for production

npm run build

The static site is generated in `dist/` and can be hosted on any static host
(Vercel, Netlify, GitHub Pages, S3, ...). For client-side routing to work,
configure the host to serve index.html for unknown paths.
